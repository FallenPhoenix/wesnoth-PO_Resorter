Wesnoth PO Sorter
~~~~~~~~~~~~~~~~~~~~~~~~~~

В игре "Battle for Wesnoth" у языковых файлов встречается 2 стандарта упорядочивания:
1) По текстовому домену с именами файлов, соответствующими коду языка. В таком виде они располагаются в исходниках, и так, вероятно, с ними удобней работать программистам.
2) По языку с именами файлов, соответствующими переводимым доменам. Так файлы располагаются в скомпилированной игре, и такой порядок явно удобней игрокам и переводчикам.
Собственно для быстрого преобразования между этими двумя видами сортировки я и написал эту утилитку.

Требования:
- MS Windows '98 либо более поздняя.
- .NET Framework 2.0
Линуксоиды могут с горем пополам юзать под Mono (уж не знаю, начиная с какой версии, но у меня там 2.10.5 стоит).

Использование:
Кнопка "Извлечь язык" - преобразование первого варианта во второй. В качестве исходной укажите базовую PO-папку.
Кнопка "Добавить язык" - наоборот из второго в первый. Исходной папкой укажите ту, где непосредственно лежат PO-файлы данного языка.
